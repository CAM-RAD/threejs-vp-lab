# Gray Box Studio

AI-assisted virtual production using a neutral gray stage instead of green screen or LED volume. Performers are captured on a gray background with tracking markers, then AI generates all environments, lighting, and set extensions in post-production.

## Reference Production

**Bitcoin: Killing Satoshi** (2026) — directed by Doug Liman, produced by Acme AI & FX. Shot entirely on a gray box stage in a converted London car showroom. $70M budget (down from $300M traditional). 20-day shoot, 30 weeks post, 55 AI artists. ~200 distinct global locations generated by AI.

## Our Setup

LED volume for hero shots with in-camera lighting. Gray box warehouse for performance capture with AI environments in post. Gaussian scanner for capturing real locations. Best of both worlds.
